# Projet Web Server
Dans le cadre du cours WebServer de troisième et denière année de Bachelor, nous avons créé une API.
Cette API est un journal de voyage.

Cette API regroupe les tables suivantes:
- user
    - username (email)
    - password
    - registration date
- trip
    - description
    - creation date
    - last modification date
- place
    - description
    - geolocalisation
    - picture
    - creation date
    - last modification date
